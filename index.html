Emphasis

To emphasize or de-emphasize text, use the following html tags:

Example:

  • <small> - For de-emphasizing inline or blocks of text, use the small tag.
  • <strong> - For emphasizing inline or blocks of text with a heavier font-weight, use the strong tag.
  • <em> - For emphasizing inline or blocks of text with italics, use the em tag.

Blockquotes

For quoting blocks of content from another source within your document. Wrap <blockquote> around any HTML as a quote.
For straight quotes we recommend a <p>
Add <small> tag for identifying the source. Wrap the name of the source work in <cite>.

Example:
